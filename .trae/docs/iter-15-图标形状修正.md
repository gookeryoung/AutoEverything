# 迭代 15：角色定位图标形状修正

## 需求清单

- [x] 修正 4 种角色定位图标的形状设计，使图标更符合语义：
  - 盾牌：从「上窄下宽 + 椭圆底」改为「上宽下尖的经典盾形轮廓」
  - 弓箭：从「只有箭头 + 箭杆」改为「弓弧 + 弓弦 + 箭杆 + 箭头」完整组合
  - 锤子铁砧：从「只有 T 字形锤子」改为「锤子 + 铁砧」组合
  - 钱袋子：从「实心圆」改为「扎口 + 袋口 + 圆形袋身」

来源：用户反馈「图标设计不正确，请完善」。

## 迭代目标

- 盾形画反了（上窄下宽，应是上宽下尖）

- 锤子铁砧只有锤子没有铁砧
- 钱袋子只是圆，不像钱袋

本次迭代重写 4 个形状判定函数，使图标视觉语义清晰可辨。

## 关键决策与依据

### 1. 盾牌改为经典盾形（上宽下尖）

**决策**：顶部水平边宽 20，中部斜边收窄（宽 20→8），底部尖角（宽 8→0）。

**依据**：
- 经典盾牌轮廓是上宽下尖（如欧洲骑士盾），原实现「上窄下宽」更像桶
- 用整数乘法比较避免浮点除法：`dx*2 <= 20-(y-8)` 等价于 `dx <= 10-(y-8)*0.5`
- 底部尖角用 `dx*7 <= (28-y)*4` 等价于 `dx <= (28-y)*4/7`

### 2. 弓箭加入弓弦让弓形完整

**决策**：弓弧（左半圆，半径 8，厚度 2）+ 弓弦（垂直线）+ 箭杆 + 箭头三角形。

**依据**：
- 原实现只有箭头 + 箭杆，缺少「弓」的元素，用户反馈「不像弓箭」
- 加入弓弦（x=10, y=8-24 垂直线）连接弓两端，箭杆从弓弦中部穿出
- 弓弧用 distSq ∈ [49, 81]（半径 7-9 的环带），厚度 2 即可，因有弓弦辅助轮廓清晰
- 用 Python 临时脚本生成 ASCII 预览验证形状（验证后删除脚本）

### 3. 锤子铁砧加入铁砧

**决策**：上方锤子（锤头 + 锤柄）+ 下方铁砧（宽顶 + 收腰 + 底座）。

**依据**：
- 原实现只有 T 字形锤子，缺少「铁砧」元素
- 铁砧形状：顶部宽平台（工作面）+ 中部收腰 + 底部宽底座（基座）
- 锤头 y=3-7 宽 16，锤柄 y=7-17 宽 2，铁砧顶 y=18-21 宽 20，铁砧腰 y=22-24 宽 10，铁砧底 y=25-28 宽 14

### 4. 钱袋子改为扎口 + 圆形袋身

**决策**：顶部扎口小三角 + 袋口收窄 + 圆形袋身。

**依据**：
- 原实现只是实心圆，无法区分「钱袋」与「硬币」
- 钱袋经典形象：顶部扎口（绳扎处）+ 袋口收窄 + 圆形膨胀袋身
- 扎口 y=2-4 小三角（`dx <= (4-y)*2`），袋口 y=5-8 宽 6，袋身 y=9-26 圆形（圆心 (16,17)，半径 10）

### 5. 用 Python 临时脚本验证像素布局

**决策**：在修改 C# 代码前，先用 Python 脚本生成 4 个图标的 ASCII 预览，验证形状正确后再同步到 C#。

**依据**：
- C# 代码需编译后才能在游戏内看到效果，反馈周期长
- Python 脚本即时输出 ASCII 图，快速迭代形状设计
- 形状判定函数是纯逻辑（输入 x,y 输出 bool），Python 与 C# 实现一致
- 验证后立即删除临时脚本，不污染仓库

## 代码实现情况

### 4 个形状判定函数最终实现

```csharp
// 盾牌：上宽下尖的经典盾形
private static bool IsShieldFilled(int x, int y)
{
    int dx = x < Center ? Center - x : x - Center;
    if (y < 4 || y > 28) return false;
    if (y <= 7) return dx <= 10;                    // 顶部水平边
    if (y <= 20) return dx * 2 <= 20 - (y - 8);     // 中部斜边
    return dx * 7 <= (28 - y) * 4;                  // 底部尖角
}

// 弓箭：弓弧 + 弓弦 + 箭杆 + 箭头
private static bool IsBowArrowFilled(int x, int y)
{
    if (y >= 15 && y <= 17 && x >= 10 && x <= 26) return true;  // 箭杆
    if (x >= 22 && x <= 28) {                                    // 箭头三角形
        int dy = y < 16 ? 16 - y : y - 16;
        if (dy <= 28 - x) return true;
    }
    if (x == 10 && y >= 8 && y <= 24) return true;              // 弓弦
    if (x >= 2 && x <= 10 && y >= 8 && y <= 24) {               // 弓弧
        int cdx = x - 10, cdy = y - 16;
        int distSq = cdx * cdx + cdy * cdy;
        if (distSq >= 49 && distSq <= 81) return true;
    }
    return false;
}

// 锤子铁砧：锤子 + 铁砧
private static bool IsHammerAnvilFilled(int x, int y)
{
    int dx = x < Center ? Center - x : x - Center;
    if (y >= 3 && y <= 7 && dx <= 8) return true;     // 锤头
    if (y >= 7 && y <= 17 && dx <= 1) return true;    // 锤柄
    if (y >= 18 && y <= 21 && dx <= 10) return true;  // 铁砧顶
    if (y >= 22 && y <= 24 && dx <= 5) return true;   // 铁砧腰
    if (y >= 25 && y <= 28 && dx <= 7) return true;   // 铁砧底座
    return false;
}

// 钱袋子：扎口 + 袋口 + 圆形袋身
private static bool IsMoneyBagFilled(int x, int y)
{
    int dx = x < Center ? Center - x : x - Center;
    if (y >= 2 && y <= 4) return dx <= (4 - y) * 2;   // 扎口小三角
    if (y >= 5 && y <= 8 && dx <= 3) return true;     // 袋口
    if (y >= 9 && y <= 26) {                          // 圆形袋身
        int dy = y - 17;
        if (dx * dx + dy * dy <= 100) return true;
    }
    return false;
}
```
