# 武器装备评分完善方案

## Context

用户要求完善武器装备评分系统，重点考虑防具护甲、防具隔温、武器伤害、武器护甲穿透四个维度，解决两个具体问题：

1. **单分子剑不被装备**：单分子剑单次伤害高、护甲穿透极高（0.83），但攻击间隔长导致 DPS 不如其他武器。当前 `WeaponDpsScorer` 只看 `MeleeWeapon_AverageDPS`，不考虑护甲穿透，导致殖民者不选单分子剑。

2. **人皮衬衫被优先选择**：人皮材质护甲极低（sharp 0.016），超织物护甲高（sharp 0.36）。当前护甲权重 200、隔温权重 10，若人皮衬衫品质远高于超织物（如传奇 vs 普通），品质加分 60 可能压过护甲优势。

## 根因分析

### 单分子剑问题

- 文件：[WeaponDpsScorer.cs](file:///e:/SteamLibrary/steamapps/common/RimWorld/Mods/AutoEverything/Source/AutoEverything/AutoEquipment/Scoring/Weapon/WeaponDpsScorer.cs)
- 近战评分只看 `MeleeWeapon_AverageDPS`，缺少护甲穿透维度
- 单分子剑 DPS 20、护甲穿透 0.83；长剑 DPS 25、护甲穿透 0.33
- 当前评分：单分子剑 100 vs 长剑 125，长剑胜出（不合理，单分子剑对高护甲敌人更有效）

### 人皮衬衫问题

- 文件：[GearWeights.cs](file:///e:/SteamLibrary/steamapps/common/RimWorld/Mods/AutoEverything/Source/AutoEverything/AutoEquipment/Scoring/GearWeights.cs)
- 护甲权重 `w_armor=200`、隔温权重 `w_insulation=10`
- 超织物衬衫护甲评分 0.405×200=81，人皮衬衫护甲评分 0.028×200=5.6
- 若人皮为传奇品质加分 60，超织物为普通品质加分 0：超织物 81 vs 人皮 65.6（当前应胜出，但差距不够大，品质差距更大时可能反转）

### API 确认

- `StatDefOf.MeleeWeapon_AverageArmorPenetration` **不存在**（编译失败）
- 正确用法：`StatDef.Named("MeleeWeapon_AverageArmorPenetration")`，返回 0~1 小数
- 远程武器护甲穿透由子弹决定，武器乘数默认 1.0，不纳入武器评分

## 改动清单

### 1. `WeaponDpsScorer.cs` — 增加近战护甲穿透评分

- 缓存 `StatDef.Named("MeleeWeapon_AverageArmorPenetration")` 到静态字段（避免 Tick 路径重复查询）
- 近战武器增加护甲穿透评分：`avgAp * weights.w_armorPenetration * contextMult`
- 战斗情境放大 1.5 倍（与 DPS 一致）
- 远程武器不变（护甲穿透由子弹决定，武器乘数默认 1.0 无差异）

**评分对比**（Standard 预设，w_dps=5, w_armorPenetration=50）：
- 单分子剑：20×5 + 0.83×50 = 100 + 41.5 = 141.5
- 长剑：25×5 + 0.33×50 = 125 + 16.5 = 141.5
- 平局，单分子剑不再被压制

### 2. `GearWeights.cs` — 调整权重

新增字段：
- `w_armorPenetration`：近战护甲穿透权重

权重调整（所有预设方案同步）：

| 字段 | Standard | Aggressive | Economic | Hunting |
|------|----------|------------|----------|---------|
| w_armor | 200→**300** | 300→**400** | 150→**250** | 100→**200** |
| w_insulation | 10→**50** | 5→**30** | 10→**50** | 10→**50** |
| w_armorPenetration | **50**（新增） | **60** | **40** | **30** |

**人皮衬衫对比**（Standard，w_armor=300）：
- 超织物衬衫（普通）：0.405×300=121.5 + 品质 0 = 121.5
- 人皮衬衫（传奇）：0.028×300=8.4 + 品质 60 = 68.4
- 超织物大幅胜出 ✓

### 3. `README.md` — 同步评分模型章节

更新权重预设方案表格、武器评分管线表格、总分公式说明，反映新增的护甲穿透维度和调整后的权重。

## 设计原则

- **最小改动**：只在 `WeaponDpsScorer` 增加护甲穿透评分，不新增 Scorer 类
- **数据驱动**：权重通过 `GearWeights` 结构体配置，所有预设方案同步调整
- **性能安全**：`StatDef` 缓存为静态字段，避免 Tick 路径重复查询
